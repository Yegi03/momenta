# Data Notes

This folder is reserved for local data links/metadata and is not used to store full datasets in git.

MOMENTA experiments in this project use the following public datasets:

- Fakeddit
- MMCoVaR
- Weibo
- XFacta

All datasets above are publicly available from their respective official releases/publication pages.

In this codebase, dataset roots are expected under:

- `datasets/Fakeddit-master/data/multimodal_only_samples/`
- `datasets/MMCoVaR-main/`
- `datasets/Weibo-dataset-main/`
- `datasets/XFacta-main/data/`
